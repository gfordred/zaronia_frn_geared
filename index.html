⚡ How To Run Locally

Place the following files in the same folder as index.html — they are loaded automatically on startup:

Required: ZARONIA_FIXINGS.csv
Optional: JIBAR_FRA_SWAPS.xlsx, target_prices.json, basis_cache.json
All files are loaded automatically from the same folder as index.html.

Start a local server (required — file:// is blocked by CORS):

python -m http.server 8000 open http://localhost:8000

Or use the VS Code Live Server extension. If JIBAR_FRA_SWAPS.xlsx is absent, market rates can be entered manually in the sidebar.
